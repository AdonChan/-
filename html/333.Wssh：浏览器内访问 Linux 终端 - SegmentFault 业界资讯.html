Wssh：浏览器内访问 Linux 终端

  • python
  • websocket
  • ssh
  • linux
4.8k 次阅读  ·  读完需要 3 分钟
2

简介

wssh 可以让我们通过 HTTP 来调用远程的一个 shell，也就是说我们可以用浏览器来访问某个 Linux 服务器／虚拟机的终端（只要这个服务器上运行了 wsshd 服务器端）。

wssh 客户端通过 ssh 帐号连接到 wsshd 服务器端。wssh 更多的是当作库来开发一些应用，比如开发云计算、虚拟机后台控制面板的虚拟机控制台等等。

这意味着，你仅仅需要几行代码，就可以为浏览器实现了一个 WSSH 客户端。

用法

尽管 wssh 最开始的目的是作为一个库用在应用程序中，它可以通过两个命令从服务器端和客户端来回切换：wsshd（服务器端）、wssh（客户端）

$ wsshd 
wsshd/0.1.0 running on 0.0.0.0:5000

$ wssh aluzzardi@mba -p
Password: 
Last login: Mon Jul 23 23:20:27 2012 from localhost
aluzzardi@mba:~$

Web 界面

wsshd 提供了一个可以连接到 Javascripts 客户端的 Web 界面

更多信息可以阅读 wssh 的 GitHub 项目主页
