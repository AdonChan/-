[译]一个程序媛的年终回顾

  • screencast
  • python

2.7k 次阅读  ·  读完需要 13 分钟

每年到了大概这个时候，我会花几天时间思考即将跨过去的这一年。常常会说些我做了什么，认识了哪些人，完成了哪些目标（当然是工作之外）。

今年没什么不同的。但我仍然觉得我可以做的更多，我不知道这是好的还是坏的反应。总之，让我们看看这一年吧。

如果你对我之前两年写的东西感兴趣的话，你可以在这儿查看。

这也是我在 Svbtle 上发的第一篇文章，让我们看看发生了什么。

进展

今年的主要目标是完成制作东西，然后卖掉它们。看着这些制作的伟大的东西，我觉得前所未有的爽，我一直喜欢了解事情是怎么在一起协同工作的和商业中的每一个面被暴露出来的实际情形。

作为一个开发者，我非常擅长产品周期和开发周期。所以今年，我的大多数精力放在学习如何营销上边，而且搞清楚了如何让人们付款和为什么人们会付款。就像我之前说的，你不亲身经历，是不可能了解事情和事情的复杂度的。本片文章全文贯彻这种思想。

Neckbeard Republic

我在这个项目上花费了大量的精力。除去写 iOS 应用之外，我一直在做屏幕录像工作，和客户打交道和给网站添加新功能。

这个概念不怎么新颖，但是作为 Python 社区中的一员，我看到了需要被弥补的空缺。所以我开始建立 Neckbeard Republic，这是一个我每周发布关于 Python，Django 主题的屏幕录像。

时光流逝，Neckbeard Republic 成立了。

I am pleased to introduce Neckbeard Republic! https://t.co/kDBSbqhFE0
— Mahdi Yusuf (@myusuf3) February 27, 2013

这些转推和来自很多人的祝福，让我感觉棒极了。

到了这一步，我应该换个角度了，我的所有注意力都放在美刀上。

开始的实际情况是这样的。在第一天结束，我有8个订阅者。我都不怎么好意思说出来，但是同时我也挺庆幸的。我一直在期待产品市场的爆发。

到了好事多磨的时候了，这是个拓展性很强的市场，所以我决定来点创意吸引和留住用户。

我决定新发一个免费的屏幕录像，这个视频接受度很高而且带来了一小部分客户。接下来几周，我又发布了好几个免费的屏幕录像，接着就看到了每周注册用户从之前的25个用户提高了250%。然而我运营的 newsletter 却没有带来多少注册用户。

I put this on in the background as I was working this morning. Someday I hope to write python tutorials this good: neckbeardrepublic.com/screencasts/__...
— Katherine A. Scott (@kscottz) Apr 11, 2011

另一个表现出来的问题，用户只注册，不订阅（订阅需要付费）

我决定采取一个疯狂的方法，和我的用户聊聊。我给所有的注册却不付费的用户每份发了一个邮件，大概有250人，询问它们，为什么不付费。

收到的反馈是无价的。令我非常震撼的是后来付费用户的数量。在这我联系的250个用户里，有26个用户打算成为付费用户。整个下午“写事实调查邮件”，看来是值了。

但是，依然有客户流失。发送这些邮件之后，我了解到与客户的沟通是吸引和留住用户的关键。

所以，我在统计所有事情上变得很严肃，活跃用户，用户观看的视频和吸引他们的有意思的话题。

一个被证明是无价的工具是 Intercom，它让我通过他们对我的生意关注度指标，来观察我的社区。活跃用户的数量，注册数量，和非活跃用户数。

它还能让用户很容易地和我一对一地沟通，这对于维持的客户群很关键。自从使用了这个功能，我的客户流失率已经锐减了70%。

对于目前这个进度我很满意，虽然不及我当时预想得那么成功。然而，它是一个有利可图的和具有教育意义的尝试。


我们第一件 Neckbeard Republic T-Shirt
感觉：平稳

iOS

我玩了一年多的 Objective-C 语言，但是从没有找到让我把它投入使用的理由。但是去年我有意识地在 App Store 发布了一个 app，并且赚一点美刀。自我委派的任务强迫我为它分配出时间。

这个目标需要我完成一下这些事情：
1. 制作一个应用
2. 营销一个应用
3. 把应用卖给别人

嗯，该怎么进行呢？我没有在 App Store 赚到一刀，也没有发推做一些推广工作，也没有给我的 Neckbeard Republic 用户发邮件。

自发布后的两周内，在 iPhone 和 iPad 上共收获了100个活跃用户。

我和 Fabien Dorion 俩人一起做这个项目，他负责这个应用的设计。

一些关于 iOS 开发的笔记。

比起来 web 开发，这太好玩了，而且只需要学一小点 stack（相对于full stack） 东西。

让你专注于你的用户所关注的，和他们想达到的目标。我的应用足够简单，提醒用户有新的屏幕录像，而且可以直接在移动设备上观看。

所以2/3还不是很糟。这巨有意思，而且我打算给移动应用加入更多奇思妙想。

感觉：非常满足

Pycoder's Weekly

我的另外一个项目，是我们（Mike 和我）从去年开始举办的 Pycoder's Weekly，是一个免费的每周 newsletter，关于每周发生的有关 Python 那些巨有趣的事情。

今年，我们向有12000个订阅用户的 newsletter，加入了一个别人可以买广告位来招聘的工作板。

我们为世界上众多的初创公司提供发布招聘帖，比如这些比较大牌的公司：Path，DataDog，Uber。

这门生意和之前的那些生意都不一样，因为它不是一个服务消费者的产品。我们直接和有比较大的预算的公司交易。

我们知道，提供用户的社会化资料表明我们可以在保密的前提下，触及那些感兴趣的开发者是至关重要的。我们的 newsletter 已经初获成功，所以我们已经有很多用户社交资料。我们需要的是被发现。

经常，像 Google 这样的招聘者已经提供了招聘 python 工作的区域，而且从那个区域找一个新的地方来放招聘列表。在 Google 中搜索关于 python 工作的结果中，前三页没有我们的页面排名。

所以我们联系了一些在前三页出现的博客的博主，说如果把我们的工作板链接到你们的博客中，那么你们找工作的时候可以找我们（有一些人很高兴这么做因为他们是 newsletter 的读者，是不是很屌？）。通过这些小技巧，我们成功地出现在了搜索关于 python 工作的第一页。

合法～

在新的一年，我们的产品会有新的设计和一些酷炫狂拽屌炸天的东西为你揭开 Python 2014大会。

感觉：乐观

演讲和大会

Youtube: How to Make Friends and Influence Developers - Mahdi Yusuf

我真的很享受演讲和起草一个演讲。它会迫使你用详细的语言说出你关于某个主题的想法，精炼你的想法到人们很容易就能听懂，而且还要对他人有用。

这篇演讲是我去年用了1/4的时间努力整理得，有关于市场和理解为什么一些人们会享受，使用，分享一些东西，而另外一些人就不会。

如果你有机会，我真的希望你可以观看这个演讲。至于说谈吐，我现在开始感觉很惬意，就和在讲台上感觉一样。我很乐意提高的是讲故事的能力，有些人可以把故事讲得很好。

感觉：愉悦

下一步

我需要一个动机，我经常都不知道时间都花哪去了。同一个心理运行轨迹吧（Running the same mental tracks）。

同样的问题，仍旧没有回答。

我会解决真实的问题，把一整年毫无限制地奉献给它，但是要知道这些会给你带来什么。
