处理缺失数据

pandas的设计目标之一就是让缺失数据的处理任务尽量轻松。

pandas使用浮点值 NaN(Not a Number) 表示浮点和非浮点数组中的缺失数据。它只是一个便于被检测出来的标记而已。

python 内置的None值也会被当做NA处理（如 string_data[0]=None）。
