接上篇

最佳在线协作工具

不管你喜不喜欢Google，你都很难否认Google在文档协作方面做得不错。多人同时编辑同一文件毫无压力。

  1. Google Docs 50.5%

  2. Google Hangout 16.3%

  3. MediaWiki 10.1%

  4. WordPress 8.8%

  5. Other 5.7%

  6. Redmine 5.3%

  7. WebODF 1.7%

  8. Feng Office 1.4%

  9. Norton Zone .1%

最佳云文件存储

dropbox始终是最多人选择的存储。开源替代ownCloud排名第二，可喜可贺。

  1. Dropbox 35.5%

  2. ownCloud 16.3%

  3. Google Drive 16.2%

  4. Ubuntu One 7.1%

  5. rsync 6.8%

  6. Other 5.2%*

  8. SpiderOak 4.6%

  12. Norton Zone .1%

最佳Linux游戏

真让我摸不着头脑。都这么多年了，Minecraft咋还这么流行呢？明年Steam系的支持Linux的游戏会不会带来改观呢？拭目以待。

  1. Minecraft 21.2%

  2. Other 20.9%*

  3. Half-Life 18.5%

  4. Frozen Bubble 11.1%

  5. Trine 2 7.1%

  6. OpenTTD 5.2%

  15. Oolite .6%

  16. BurgerSpace .4%

最佳显卡品牌

NVIDIA以其强劲的性能和对Linux的良好支持拔得头筹，Intel的集成显卡性能较弱，但是其Linux下的开源驱动口碑良好，因此获得第二。

  1. NVIDIA 51.6%

  2. Intel 26.3%

  3. AMD 21.3%

  4. Other .8%

最佳SQL数据库

SQL数据库可能不那么激动人心，但是它非常重要。MySQL仍旧是第一名，但是它的分支MariaDB追得很紧。PostgreSQL同样表现不错。

  1. MySQL 29.7%

  2. MariaDB 28.5%

  3. PostgreSQL 26.8%

  4. SQLite 8.8%

  5. Oracle 4.4%

  6. Other 1.8%

最佳NoSQL数据库

NoSQL发展迅猛。MongoDB以超过40%的得票率排名第一。

  1. MongoDB 43.6%

  2. Apache HBase 15.3%

  3. Cassandra 13.2%

  4. CouchDB 13.2%

  5. Other 7.7%

  6. Redis RethinkDB 4.5%

  7. Neo4j 2.6%

最佳备份解决方案

其实小编不在乎你选择哪个方案，重要的是，一定要备份！有趣的是，Dropbox在备份方案中排名第二。虽然Dropbox有版本功能，但是小编以前没想过它是备份方案。小编老啦……

  1. clonezilla 19.6%

  2. Dropbox 19.3%

  3. Other 19.1%*

  4. Bacula 8.8%

  5. rdiff-backup 7.7%

  6. CrashPlan 7.5%

  7. Back In Time 5.4%

  8. Amanda 4.5%

  9. luckyBackup 4.1%

  10. Tivoli Storage Manager 1.8%

  11. Symantec Backup Exec 1.4%

  12. Storix .5%

  13. Areca-Backup .4%

最佳虚拟化解决方案

很高兴看到开源的方案占据前两名。KVM得票稍低，可能是因为它只支持Linux，而不像VirtualBox一样跨平台吧。

  1. Oracle VM VirtualBox 37%

  2. KVM 22.2%

  3. VMware 21.2%

  4. XEN 4.9%

  6. QEMU 4.1%

  9. Linux-VServer 1.3%

  10. Symantec Workspace Virtualization .2%

  1. Nagios 21.9%

最佳开源配置管理工具

这里的结果和小编的期望差距不小。虽然小编爱死apt了，但是从来不认为它是一个配置管理工具。还有人用Subversion管理配置，真是高大上的选择。

  1. apt 38.8%

  2. Puppet 21.1%

  3. Yast 11.6%

  4. Other 9.6%*

  5. Subversion 9.2%

  6. SaltStack 6.6%

  7. CFEngine 3.1%

最佳包管理

前两名都是管理deb包的工具，pacman紧随其后。有一半以上的用户投给了apt或Synaptic，但是投给pacman的13%的用户说明Arch Linux变得流行了。

  1. apt-get 38.5%

  2. Synaptic 13.7%

  3. pacman 12.9%

  4. RPM 11.1%

  5. Aptitude 6.9%

  6. Other 6.2%*

  7. Gentoo Portage 4.3%

  9. dpkg 1.5%

  10. SaltStack 1.4%

  11. pkgtool 1.1%

  12. Nix .5%

  13. Conary .3%

最佳版本控制系统

接近80%的投票，git是当之无愧的王者。

  1. Git 78.3%

  2. Subversion 11.8%

  3. Mercurial 4.9%

  4. Bazaar 2.6%

  5. Other 1.8%

  6. Plastic SCM .7%

最佳调试器

老牌调试器GDB获得四分之三强的投票。由于Python的流行，PuDB位居第二。

  1. GDB (The Gnu Project Debugger) 75.8%

  2. PuDB (Python Debugger) 13.9%

  3. Other 6.5%

  4. Edebug 3.8%

最佳开源安全工具

nmap获得第一——你还记得哪部电影里黑客的电脑上显示的是nmap的源码么？排名第二、第三的BackTrack Linux和Kali Linux都是专注安全的Linux发行版。密码管理工具KeePass也得到不少投票。

  1. Nmap 23.2%

  2. BackTrack Linux 17.3%

  3. Kali Linux 15%

  4. KeePass 14.6%

  5. Netfilter 7.8%

  6. pfSense 6.9%

  7. Metasploit 6.5%

  8. Other 5%

  9. AIDE 2.3%

  10. OSSEC 1.5%

最佳开源渗透测试工具

nmap再次夺得第一。Kali Linux也获得了第二，它对排名第三的Metasploit支持良好。

  1. Nmap 37.1%

  2. Kali Linux 32.3%

  3. Metasploit 25.1%

  4. Other 5.6%
