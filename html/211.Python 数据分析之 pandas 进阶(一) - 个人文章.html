导入本篇中使用到的模块：

    import numpy as np
    import pandas as pd
    from pandas import Series, DataFrame

我们可以调整数据输出框大小以便观察：

    pd.set_option('display.width', 200)

一、创建对象
1、可以通过传递一个list对象来创建一个Series，pandas会默认创建整型索引：

    s = pd.Series([1,3,5,np.nan,6,8])
    s
    0     1
    1     3
    2     5
    3   NaN
    4     6
    5     8
    dtype: float64
